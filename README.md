
# 📘 dsa01

**A web platform to learn and practice Data Structures and Algorithms (DSA)**
🌐 Live site: [https://dsa01.vercel.app](https://dsa01.vercel.app)

---

## 📌 Overview

**dsa01** is a modern, browser-based application designed to help students, developers, and coding enthusiasts explore core Data Structures and Algorithms in an interactive and engaging way. Whether you're revising for interviews or learning DSA for the first time, this tool aims to make the experience intuitive and visual.

---

## 🚀 Features

* **Interactive Visualizations of the progress**  
* **progress tracking**
* **Clean & Responsive UI**

---

## 🛠️ Technologies Used

* **Frontend**: React, JavaScript, HTML5, CSS3
* **Backend**: Nodejs, express
* **Deployment**: Vercel
